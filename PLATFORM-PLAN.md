## 🧱 الستاك (للتوسّع)
**Astro + TypeScript + Cloudflare Pages + Pagefind (بحث الموقع) + MiniSearch (بحث الكتاب) + PWA + i18n (ar/en/cop).** بدون backend.
- ليه: المحتوى ٩٥٪ نص ساكن؛ Astro بيطلّع zero-JS وأفضل SEO؛ Cloudflare اشترت Astro (يناير ٢٠٢٦) + bandwidth غير محدود + أقرب edge للـMENA. **مش Next.js** (تعقيد + تسعير مش محتاجينه) ولا SvelteKit.
- الهجرة = **port مش rewrite**: ننقل المنطق الشغّال (جدول الأسفار، normalization، parseRef، VOTD، theme/font، والـdaily-readings Action) كما هو.
- موبايل: **PWA دلوقتي**، و**Capacitor** لاحقاً يلفّ نفس كود الويب للـApp Store — **من غير React Native/Flutter أبداً**.

## 🔎 البحث + «اسأل الأرثوذكسية» (حيطة صارمة بين مرحلتين)
- **Phase A (أيام، مجاني، صفر خطر):** نستبدل `searchCached` (بيدوّر في اللي قريته بس) بـ**بحث كامل للكتاب** عبر index ساكن مبني وقت الـCI فوق ~٣١ ألف آية بـ**MiniSearch** + **دالة Arabic normalization واحدة** (شيل التشكيل/التطويل، وحّد أ/إ/آ→ا، ى→ي، ة→ه، أرقام عربية→لاتيني) تتطبّق على النص والـquery بنفس الشكل — **دي اللي بتخلّي البحث العربي يشتغل صح**.
- **Phase B (شهور، opt-in، بعد يوليو):** RAG **بمصادر مُتحقَّقة واستشهاد إجباري** — **مش chatbot حر**. hybrid retrieval (BM25 + dense عبر bge-m3/e5 على **Odysseus**) فوق corpus **قانوني نظيف بس**، prompt «جاوب من المصادر وإلا قول مش عارف واستشر أب اعترافك»، عتبة ثقة بترفض، **تحقّق من الاقتباس**، وإطلاق behind-a-flag لكام كاهن/خادم مع زر «بلّغ عن إجابة».
> 🕯️ القاعدة المقدّسة: **آية مهلوسة أو هرطقة = نهاية المشروع.** الـrefuse-by-default غير قابل للتفاوض.

## 🏆 ليه نقدر نتفوّق على St-Takla
مش بكمية المحتوى (عندهم DR≈63 و~182 ألف زيارة/شهر و٢٠ سنة) — لكن بـ: **السرعة + الموبايل + التصميم** (هما أرشيف بطيء ٢٠١٠)، و**الظهور في محركات الإجابة** (St-Takla والتطبيقات **خُفاة على ChatGPT/Perplexity** — لين مفتوح بالكامل)، و**الـAI المستشهد بالمصادر** (محدش في القبطي عنده). خطة AEO = صفحات Q&A منظمة (FAQPage schema) + إجابة مباشرة ٤٠–٦٠ كلمة + سماح لـGPTBot/PerplexityBot/ClaudeBot.
**الربح:** وقف/تبرعات (Zeffy صفر-عمولة + Paymob/Fawry/InstaPay لمصر + Stripe/PayPal للمهجر) — **لا إعلانات ولا paywall على الكتاب**.
**التشغيل:** CMS إلزامي قبل المتطوعين (Sanity أو Decap/Keystatic) + دور «مساهم» يكتب draft، كاهن/محرّر يعتمد = بوابة ضد الهرطقة.
