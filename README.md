# الكتاب المقدّس — النسخة الأرثوذكسية (قارئ أونلاين)

قارئ كتاب مقدّس عربي حديث، سريع، RTL، بوضع ليلي/نهاري. بديل أنظف وأسرع من st-takla
على محور التصميم وتجربة القراءة.

## الملفات
- `index.html` — الهيكل
- `styles.css` — التصميم (بيزنطي/طقسي: ذهبي + رماد ليلي + ورقي نهاري)
- `app.js` — الكانون الكامل + التحميل من الـAPI + البحث + آية اليوم + حفظ آخر قراءة

## شغّال إزاي
- **النص:** ترجمة فان دايك (مجال عام) من `api.getbible.net` (مفتاح `arabicsv`).
  بيتحمّل **مباشرةً في المتصفح** ويتخزّن مؤقتاً في `localStorage` — احنا مش بنستضيف النص ولا بنوزّعه.
- **بدون بناء (no build):** ملفات ساكنة. أي خادم ساكن يشغّلها.

## تشغيل محلي
```bash
cd ~/Documents/Coptic-Bible-Web
python3 -m http.server 8150
# افتح http://127.0.0.1:8150
```
> ملاحظة ماك: خادم Claude للمعاينة مش بيقرأ من `~/Documents` (حماية TCC)،
> فالنسخة الحيّة بتتخدم من `/tmp/coptic_bible_site`. الأصل هنا في Documents.

## اللي شغّال دلوقتي ✅
- ٦٦ سفر كاملين (عهد قديم + جديد) بالنص الحقيقي
- فهرس أسفار مقسّم بالعهود + فلترة
- قارئ بأرقام آيات + تنقّل إصحاحات + لمس الآية تنسخها/تشاركها
- بحث وانتقال بالشاهد (مثال: «يوحنا ٣:١٦») + بحث نصّي داخل المقروء
- آية اليوم (بتتغيّر يومياً) + آية عشوائية + متابعة آخر قراءة
- ليلي/نهاري + تكبير الخط (محفوظين)

## الخطوة الجاية (لو كمّلنا) 🔜
1. **الأسفار القانونية الثانية** (طوبيا/يهوديت/الحكمة/سيراخ/المكابيين…): مدرجة في الفهرز
   ومعلّمة "قريباً" — محتاجة نص من نسخة أرثوذكسية/كاثوليكية عربية (فان دايك بروتستانتي بلا أبوكريفا).
2. بحث شامل على الكتاب كله (فهرسة مسبقة بدل البحث داخل المقروء).
3. صوت/تلاوة، تفاسير آبائية، خطة قراءة يومية، مشاركة كصورة آية.
4. PWA (يشتغل أوفلاين بعد أول فتح) + دومين خاص.
